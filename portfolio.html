Портфолио

В рамках сотрудничества с компанией разработчик 3D приложения в области QA автоматиации были разработаны следующие приложения:

Фреймворк: "AdminDatabase"

Системные требования:
Apache (version: 2.4.23 и выше.), MySQL (version: 5.7.14 и выше.), PHP (version: 5.6.25 и выше.)

Фреймворк (платформа) AdminDatabase позволяет создать администратор баз данных для Веб-приложения на основе Apache, MySQL с использованием языков программирования PHP и Javascript. Подобная "админка" часто используется для работы с данными в играх. Вы можете разработать свой администратор на основе данного фреймворка с последующим развертыванием на хостинге. Такая "админка" предоставит возможность работать с данными без использования phpmyadmin.

Программа: "FastADB" version 1.0

Системные требования: Windows, Android Studio (Android SDK), Jython.

Программа FastADB позволяет работать с Android Debug Bridge более удобно и быстрее. С помощью этой программы вы легко сможете снимать логи и скриншоты с устройства, устанавливать и удалять приложения, производить стресс-тестирвоание, и многое другое.

Программа: "FastADB" version 1.1

Системные требования: Windows, Android Studio (Android SDK)

Вторая облегченная версия программы FastADB. Для этой программы не требуется установка Jython и использование дополнительных средств тестирования. Использование этой программы максимально упрощено при этом сохранены все самые необходимые функции, такие как: снятие логов, запись видео с экрана, скриншот с экрана, установка приложения, стресс-тестирование и прочее.
