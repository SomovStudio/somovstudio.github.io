О проекте

Проект Somov Studio основан независимым разработчиком программного обеспечения с открытым исходным кодом. Автор проекта Сомов Евгений Павлович является специалистом в области автоматизированного тестирования и занимается оказанием услуг в облати тестирования программного обеспечения.

Автоматизация процессов тестирования неотъемлемая часть разработки любого программного обеспечения, она позволяет сократить время и вероятность пропущенных ошибок во время выполнения регресс-тестировании, а так же упрощает выполение сложных тестов.

Разработка специализированного программного обеспечения предназначенного для упрощения выполнения задачь в области тестирования. На данном ресурсе вы найдете утилиты помогающие в тестировании. Все предлагаемые программы с открытым исходным кодом были разработаны автором проекта и распространяются абсолютно бесплатно.

Программное обеспечение

Ведется разработка программного обеспечения предназначенного для упрощения выполнения задачь в области тестирования. Все предлагаемые программы с открытым исходным кодом и распространяются абсолютно бесплатно.

  • FastADB - упращает работу с Android Debug Bridge.
  • DetectGaYm - тестирование событий Google analytics и Yandex metrika.
  • SearchSiteContent - поиска контента на страницах из sitemap
  • SEOScanner - программа сканирования web страниц для проведения SEO-аудита
  • TestSitemap - тестирование ссылок прописанных в файле sitemap.
  • TestCookie - тестирования cookie.
  • TestRedirect - тестирование редирект ссылок.
  • JsonFileEditor - редактирование json файлов.
  • Admin Database Framework - фреймворк для создания админок.

* Все прогреммы находятся в открытом доступе по лицензии GNU и вы можете скачать их совершенно бесплатно.
Нажмите "подробнее" чтобы ознакомиться с описание по каждой программе.

Технологии

Основные технологии автоматизации процессов тестирования Web и Mobile приложений: Selenium, Appium
Выполнение модульного тестирования средствами: JUnit, TestNG, PHPUnit, NUnit, Unittest, PyTest
Самые популярные фреймворки автоматизированного тестирования Web приложений: Codeception, Selenide, Cucumber, WebdriverIO, Robot, Cypress
Стеки автоматизации тестирования Web приложений:

  • Selenium + Codeception + PHPUnit (PHP)
  • Selemiun + JUnit/TestNG (Java)
  • Selenium + Unittest/PyTest (Python)
  • Selenium + NodeJS (JavaScript)
  • Selenide + Cucumber + TestNG (Java)
  • Robot framework (Python)
  • WebdriverIO (JavaScript)
  • Cypress (JavaScript)
Стеки автоматизации тестирования Mobile приложений:
  • Appium + UIAutomator2 + JUnit/TestNG (Java)
  • Unity3D + Test Runner (C#)
Комплексные инструменты для автоматизации тестирования Web и Mobile приложений: Katalon Studio, TestComplete
Интеграция автотестов в Jenkins и TeamCity а так же GitLab Runner
