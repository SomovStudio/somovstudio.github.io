О компании

Наша компания занимается тестированием программного обеспечения любой сложности. В работе применяется автоматизированное и ручное тестирование. Тестирование 2D и 3D приложений разработанных под мобильные устройства iOS и Android, а так же desktop и browser версии. Тестирование Web-приложений с применением автоматизации.

Так же наша компания занимается разработкой клиент-серверных приложений на основе технологий разработанных корпорацией Microsoft, таких как .NET и система управления базами данных SQL Server. Разработка web-приложений на основе технологий Apache, PHP, MySQL. Проектирование баз данных. Разработка вспомогающих утилит.

Технологии

В тестировании используются следующие технологии автоматизации: Codeception, Selenium, Katalon Studio, Unity Test Runner, Appium.

Выполнение модульного тестирования средствами: XUnit, NUnit, JUnit, PHPUnit.


В разработке клиент-серверного программного обеспечения используется .NET Framework одну из самых передовых технологии от корпорации Microsoft, и система управления реляционными базами данных Microsoft SQL Server.

В разработке web-приложений используется технология Apache, PHP и система управления реляционными базами данных MySQL.
