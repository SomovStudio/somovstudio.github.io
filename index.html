О проекте

Проект Somov Studio основан независимым разработчиком программного обеспечения с открытым исходным кодом. Автор проекта Сомов Евгений Павлович является специалистом в области автоматизированного тестирования и занимается оказанием услуг в облати тестирования программного обеспечения.

Автоматизация процессов тестирования неотъемлемая часть разработки любого программного обеспечения, она позволяет сократить время и вероятность пропущенных ошибок во время выполнения регресс-тестировании, а так же упрощает выполнение сложных тестов.

Ведется разработка специализированного программного обеспечения предназначенного для упрощения выполнения задач в области тестирования. На данном ресурсе вы найдете утилиты помогающие в тестировании. Все предлагаемые программы с открытым исходным кодом были разработаны автором проекта и распространяются абсолютно бесплатно.

Разработки

В рамках данного проекты было разработано программное обеспечение которое предназначено для упрощения выполнения задач в области тестирования.

  • Hat - браузер со встроенной технологией автоматизации тестирования Web приложений.
  • SearchSiteContent - поиска контента на страницах из sitemap
  • SEOScanner - программа сканирования web страниц для проведения SEO-аудита
  • TestSitemap - тестирование ссылок прописанных в файле sitemap.
  • TestCookie - тестирования cookie.
  • TestRedirect - тестирование редирект ссылок.
  • DetectGaYm - тестирование событий Google analytics и Yandex metrika.
  • FastADB - упращает работу с Android Debug Bridge.
  • JsonFileEditor - редактирование json файлов.
  • Admin Database Framework - фреймворк для создания админок.

* Все программы находятся в открытом доступе и вы можете скачать их совершенно бесплатно.
Нажмите "подробнее" чтобы ознакомиться с описание по каждой программе.

Технологии

Основные технологии автоматизации процессов тестирования Web и Mobile приложений: Selenium, Appium

Самые популярные фреймворки автоматизированного тестирования Web приложений: Codeception, Selenide, Cucumber, WebdriverIO, Robot, Cypress

Модульное тестирование средствами: JUnit, TestNG, PHPUnit, NUnit, Unittest, PyTest

Комплексные инструменты для автоматизации тестирования Web и Mobile приложений: Katalon Studio, TestComplete

Интеграция автотестов в Jenkins и TeamCity а так же GitLab Runner
