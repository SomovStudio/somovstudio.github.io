Программа: "Rapid Sales"

Системные требования: Windows, Microsoft SQL Server

Бухгалтерская программа "Rapid Sales" предназначена для учета продаж товаров со склада. Учет ведется по нескольким направлениям: бухгалтерский учет и складской учет. Так же в программе доступно формирование отчетов, таких как остатки ТМЦ, оборотно сальдовая ведомость по счету, по контрагентам, по торговым представителям.
