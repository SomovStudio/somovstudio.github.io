Программное обеспечение

В данном разделе вы можете скачать совершенно бесплатно разработанное нами следующее программное обеспечение:



Программа: "FastADB"

Системные требования: Windows, Android Studio (Android SDK)

Программа FastADB позволяет работать с Android Debug Bridge более удобно и быстрее. С помощью этой программы вы легко сможете снимать логи и скриншоты с устройства, устанавливать и удалять приложения, производить стресс-тестирование, и многое другое.

Программа: "Json File Editor"

Системные требования: Windows 7, 8, 10 (.Net Framework 2.0)

Программа предназначена для редактирования json файлов. В данной программе используется дерево объектов для навигации, текстовое поле для общего представления содержания файла, таблица для непосредственного редактирования данных в json файле.

Фреймворк: "AdminDatabase"

Системные требования:
Apache (version: 2.4.23 и выше.), MySQL (version: 5.7.14 и выше.), PHP (version: 5.6.25 и выше.)

Фреймворк (платформа) AdminDatabase позволяет создать администратор баз данных для Веб-приложения на основе Apache, MySQL с использованием языков программирования PHP и Javascript. Подобная "админка" часто используется для работы с данными в играх. Вы можете разработать свой администратор на основе данного фреймворка с последующим развертыванием на хостинге. Такая "админка" предоставит возможность работать с данными без использования phpmyadmin.
