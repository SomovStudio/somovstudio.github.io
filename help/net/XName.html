Создание имен с помощью XName

Как упоминалось ранее в главе, в LINQ to XML нет необходимости непосредственно

создавать имена с объектами XName. Фактически класс XName не имеет общедоступных

конструкторов, так что нет способа создавать его экземпляры. Объект XName может

быть создан из строки с необязательным пространством имен автоматически при

создании объекта XName.
