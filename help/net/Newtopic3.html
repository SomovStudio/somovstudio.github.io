Строки

Построение строк


string str = "Строки в C# весьма эффективны";


Строку можно создавать из массива символов char


char[] chArray = { 't','e','s','t' };
string strTest = new string(chArray);


Обращение со строками


Compare - сравнение строк. Если строка strA меньше строки strB; положительное значение, если строка strA больше строки strВ; и нуль, если сравниваемые строки равны.)


int resultCompare = string.Compare(strA, strB);


Equals - сравнение строк. Возвращает логическое значение true, если вызывающая строка имеет такое же значение, как и у аргумента value.


bool resultEquals = string.Equals(strA, strB);


IndexOf - Осуществляет поиск в вызывающей строке первого вхождения символа, определяемого аргументом value.


int resultIndex = strA.IndexOf('a');
resultIndex = strB.IndexOf("b", StringComparison.CurrentCulture);


LastIndexOf - Осуществляет поиск в вызывающей строке последнего вхождения символа, определяемого аргументом value.


int resultLastIndex = strA.LastIndexOf('a');


ToLower - Возвращает вариант вызывающей строки в нижнем регистре.

ToUpper - Возвращает вариант вызывающей строки в верхнем регистре.


string resultString = strA.ToLower();
resultString = strB.ToUpper();


Массивы строк


string[] strArray = { "Hello", " ", "World", " ", "!!!" };
foreach(string value in strArray)
{
    Console.Write(value);
}


Подстроки Substring


string orgstr = "В C# упрощается обращение со строками.";
string substr = orgstr.Substring(5, 20);


Применение строк в операторах switch


string[] strs = { "один", "два", "три" };
foreach(string value in strs)
{
    switch(value) {
        case "один":
            Console.WriteLine("один");
            break;
        case "два":
            Console.WriteLine("два");
            break;
        case "три":
            Console.WriteLine("три");
            break;
    }
}
