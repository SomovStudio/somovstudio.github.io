DefaultIfEmplty

LINQ ›› LINQ to Objects ›› Отложенные операции ››

DEFAULT IF EMPTY - если последовательность пуская, то возвращает последовательность со значениями по умолчанию


string[] users = {"John", "Jackson", "Bill", "Admin", "User", "sa"};
string value = users.Where(u => u.Equals("Smith")).DefaultIfEmpty().First();
if (value == null) Console.WriteLine("Пользователь Smith не найден");
