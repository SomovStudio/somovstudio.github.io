Count, LongCount

LINQ ›› LINQ to Objects ›› Не отложенные операции ››

COUNT

Возвращает количество элементов во входной последовательности


string[] users = {"Admin", "User", "sa", "John", "Jackson", "Bill"};
int count = users.Count();
Console.WriteLine(count);


В результате:


LONG COUNT

Возвращает количество элементов входной последовательности как значение типа long


string[] users = {"Admin", "User", "sa", "John", "Jackson", "Bill"};
long count = users.LongCount();
Console.WriteLine(count);
