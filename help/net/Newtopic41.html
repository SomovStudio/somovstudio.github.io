Списки и коллекции

Класс List - неупорядоченный, последовательный список элементов любого одного типа данных.


List<int> dataList = new List<int>();
for(int i = 0; i < 5; i++)
{
    dataList.Add(i);
}

foreach(int value in dataList)
{
    Console.WriteLine("dataList значение: " + value);
}

Класс Dictionary - позволяет получить мгновенный доступ к элементу по его ключу


Dictionary<string, int> data = new Dictionary<string, int>();    // key, value

data.Add("hundred", 100);
data.Add("thousand", 1000);
data.Add("million", 1000000);

Console.WriteLine("million значение = " + data["million"]);
Console.WriteLine("hundred значение = " + data["hundred"]);
Console.WriteLine("thousand значение = " + data["thousand"]);

Класс Stack - особый вид списка, основанный на модели LIFO (последним вошел - первым вышел)


Stack<int> data = new Stack<int>();
for(int i = 0; i < 5; i++)
{
    data.Push(i);
}

while(data.Count > 0)
{
    Console.WriteLine("Стэк значение: " + data.Pop());
}
