Виртуальные методы и их переопределение

Метод объявляется как вирутальный в базовом класса с помощью virtual


class BaseClass
{
    public virtual void Show()
    {
        Console.WriteLine("Метод Show в базовом классе BaseClass");
    }
}

class TestClass: BaseClass
{
    public override void Show()
    {
        Console.WriteLine("Метод Show в производном классе TestClass");
    }
}


Пример использования


BaseClass bc = new BaseClass();
bc.Show();

TestClass test = new TestClass();
test.Show();
