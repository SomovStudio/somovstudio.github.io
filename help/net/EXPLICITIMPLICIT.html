explicit , implicit операторы преобразования

Если оператор преобразования указан в неявной форме (implicit), то преобразование вызывается автоматически, например, в том случае, когда объект используется в выражении вместе со значением целевого типа.


Если же оператор преобразования указан в явной форме (explicit), то преобразование вызывается в том случае, когда выполняется приведение типов.


Преобразование с помощью implicit (неявное преобразование)


class Points3D
{
    private int x, y, z;
    public Points3D(){ x = y = z = 0; }
    public Points3D(int X, int Y, int Z){ 
        x = X;
        y = Y;
        z = Z;
    }
    
    // Неявное преобразование объекта типа Points3D к типу int
    public static implicit operator int(Points3D op)
    {
        return op.x + op.y + op.z;
    }
    
    public void Show(string msg)
    {
        Console.WriteLine(msg + ": x=" + x + " y=" + y + " z=" + z);
    }
}


Использоание преобразования


Points3D points = new Points3D(1, 2, 3);
int i;

i = points; // преобразовать в тип int

points.Show("Содержимое points");
Console.WriteLine("Результат присвоения (i = points): " + i);

Преобразование с помощью explicit (явное преобразование)


class Points3D
{
    private int x, y, z;
    public Points3D(){ x = y = z = 0; }
    public Points3D(int X, int Y, int Z){ 
        x = X;
        y = Y;
        z = Z;
    }
    
    // Явное преобразование объекта типа Points3D к типу int
    public static explicit operator int(Points3D op)
    {
        return op.x + op.y + op.z;
    }
    
    public void Show(string msg)
    {
        Console.WriteLine(msg + ": x=" + x + " y=" + y + " z=" + z);
    }
}


Использоание преобразования


Points3D points = new Points3D(2, 4, 6);
int i;

i = (int) points; // явное преобразовать в тип int

points.Show("Содержимое points");
Console.WriteLine("Результат присвоения (i = points): " + i);
