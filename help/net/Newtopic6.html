Двумерный массив


int[,] table = new int[10,20];
table[3,5] = 10;

for (var i = 0; i < 10; i++)
{
    for (var j = 0; j < 20; j++)
    {
        table[i,j] = i * j;
        Console.WriteLine("table[" + i + "," + j + "]:" + table[i,j]);
    }
}

Возврат массива из метода


class Matrix
{
    public int[,] createMatrix3x3()
    {
        int[,] matrix = new int[3,3];
        for(int i = 0; i < 3; i++)
        {
            for(int j = 0; j < 3; j++)
            {
                if(i == j) matrix[i, j] = 1;
                else matrix[i, j] = 0;
            }
        }
        return matrix;
    }
}



Matrix m = new Matrix();
int[,] matrix = m.createMatrix3x3();

for(int i = 0; i < matrix.Length / 3; i++)
{
    Console.WriteLine(matrix[i,0] + " " + matrix[i,1] + " " + matrix[i,2]);
}


В результате будет получен массив
