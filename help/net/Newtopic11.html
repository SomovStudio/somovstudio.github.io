Именованные аргументы

Именование аргументов дает возможность передавать данные в метод в любом порядке. [аргумент : значение]


public class NamedArgsDemo
{
    public static bool IsFactor(int val, int divisor)
    {
        if((val % divisor) == 0) return true;
        return false;
    }
    
    public static void demo()
    {
        bool result;
        result = IsFactor(10, 2);
        result = IsFactor(val: 10, divisor: 2);
        result = IsFactor(divisor: 2, val: 10);
    }
}
