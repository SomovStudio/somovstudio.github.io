SEALED Предотвращение наследования

C# ›› Наследование ››

Предотвращение наследования с помощью ключевого слова sealed


class ClassA
{
    public virtual void Show()
    {
        Console.WriteLine();
    }
}

class ClassB : ClassA
{
    sealed public override void Show()
    {
        Console.WriteLine();
    }
}
