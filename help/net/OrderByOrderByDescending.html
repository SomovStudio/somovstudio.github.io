OrderBy, OrderByDescending

LINQ ›› LINQ to Objects ›› Отложенные операции ››

ORDER BY - упорядочить последовательность


string[] users = {"John", "Jackson", "Bill", "Admin", "User", "sa"};
IEnumerable<string> values = users.OrderBy(u => u.Length);
foreach(var value in values)
{
    Console.WriteLine(value);
}

ORDER BY DESCENDING - упорядочивание по убыванию


string[] users = {"John", "Jackson", "Bill", "Admin", "User", "sa"};
IEnumerable<string> values = users.OrderByDescending(u => u);
