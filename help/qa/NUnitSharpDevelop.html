NUnit (SharpDevelop)

1. Создайте новое Windows-приложение


2. Добавьте к проекту Модульный тест. Для этого откройте контекстное меню -> Добавить -> Новое...



Выделите папку C# и выберите Модульный тест.




3. Создайте пустой класс и опишите в нем простую проверку email адреса


using System;

namespace TestNUtin
{
    /// <summary>
    /// Description of EmailAddress.
    /// </summary>
    public class EmailAddress
    {
        String emailAddress;
        
        public EmailAddress(String email)
        {
            emailAddress = email;
        }
        
        public bool IsValid()
        {
            if (emailAddress.Contains("@")){
                return true;
            }
            return false;
        }
    }
}



4. Откройте ранее созданный Модульный тест и опишите короткую проверку


using System;
using NUnit.Framework;

namespace TestNUtin
{
    [TestFixture]
    public class Test1
    {
        [Test]
        public void TestMethod()
        {
            EmailAddress email = new EmailAddress("test@example.com");
            bool isValid = email.IsValid();
            
            Assert.IsTrue(isValid);
        }
    }
}



5. Откройте окно инструментов для тестирования. Для этого откройте меню Вид -> Инструменты -> Тесты




6. Запустите тестирование нажатием на кнопку Запустить тест



В результате проверки вы получите сообщение. В окне Тесты отмечаются все успешные шаги зелёным цветом.



Если тест не проходит это отображается в окне Тесты красным цветом



И выводится сообщение об ошибке тестирования
