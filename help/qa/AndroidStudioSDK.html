Установка Android Studio (SDK)

Android SDK входит в состав Android Studio которое можно скачать по адресу: https://developer.android.com/studio/index.html


1. Скачайте и установите Android Studio по адресу: C:\Program Files\Android\Android Studio


2. Установите необходимые пакеты через SDK Manager


Чтобы открыть окно SDK Manager необходимо в начальном окне открыть меню Configure -> SDK Manager



Далее необходимо установить следующие пакеты:

Android SDK Build-Tools

Android Emulator

Android SDK Planform-Tools

Android SDK Tools

Google USB Driver

Google Web Driver

и т.д. (смотрите картинку ниже)



Данные пакеты устанавливаются по адресу: C:\Users\###\AppData\Local\Android


3. После завершения установки пакетов необходимо прописать адреса в системной переменной Path


Откройте свойство системы -> Дополнительные параметры -> Параметры среды



Найдите переменную Path в окне "Системные параменные"



Пропишите в этой переменной следующие адреса:


C:\Users\###\AppData\Local\Android\sdk;

C:\Users\###\AppData\Local\Android\sdk\build-tools\25.0.3;

C:\Users\###\AppData\Local\Android\sdk\platform-tools;

C:\Users\###\AppData\Local\Android\sdk\tools;
