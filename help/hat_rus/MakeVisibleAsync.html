MakeVisibleAsync

Описание: метод делает элемент видимым путем добавления стилей

Синтаксис: MakeVisibleAsync(string by, string locator, string visibility = "visible", int opacity = 1, int index = 1000)


Пример:

await tester.MakeVisibleAsync(Tester.BY_XPATH, "//*[@id='button']");


await tester.MakeVisibleAsync(Tester.BY_BY_CSS, "//*[@id='button']");
