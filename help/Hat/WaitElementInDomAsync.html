WaitElementInDomAsync

Описание: метод выполняет временную остановку выполнения теста на указанное количество секунд и ждет когда запрашиваемый элемент появится в DOM

Синтаксис: WaitElementInDomAsync(string by, string locator, int sec)


Пример:

await tester.WaitElementInDomAsync(Tester.BY_XPATH, "//div[@id='result']", 5);

await tester.WaitElementInDomAsync(Tester.BY_CSS, "#result", 5);
