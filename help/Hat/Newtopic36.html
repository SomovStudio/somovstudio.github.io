Отчет работы автотеста


При запуске автотеста в папке проекта будет создана новая папка reports если её еще нет.

Так же будет создан файл отчета в пормате html.

Имя файла отчета соответствует имени файла автотеста с приставкой Report

Например при выполнении автотеста ExampleTest1.cs будет создан отчет с именем Report-ExampleTest1.html
