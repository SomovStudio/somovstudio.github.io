CreateHashMD5FromTextAsync

Описание: метод создает HashDM5 код из указанного текста

Синтаксис: CreateHashMD5FromTextAsync(string text)


Пример:

string text = "Hello World";

string hash = await tester.CreateHashMD5FromTextAsync(text);

tester.ConsoleMsg(hash);
