ScrollToElementAsync

Описание: метод выполняет прокрутку к указанному элементу (параметр behaviorSmooth определяет плавность прокрутки)

Синтаксис: ScrollToElementAsync(string by, string locator, bool behaviorSmooth = false)

Возвращаемое значение: отсутствует


Пример:

await tester.ScrollToElementAsync(Tester.BY_CSS, "body > footer", true);


await tester.ScrollToElementAsync(Tester.BY_XPATH, "/html/body/footer", true);
