AssertNetworkEventsAsync

Описание: метод выполняет проверку присутствия (presence = true) или отсутствие (presence = false) указанных событий (events),  данный метод хорошо использовать при проверке событий Google Analytics и Яндекс Метрика

Синтаксис: AssertNetworkEventsAsync(bool presence, string[] events)


Пример:

await tester.AssertNetworkEventsAsync(true, new string[] { 

       "ec=zayavka", "ea=b2c_new_main", "el=some_shpd_nm", "zayavka_shpd" 

});
