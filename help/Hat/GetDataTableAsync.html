GetDataTableAsync

Описание: метод возвращает таблицу записей из таблицы базы данных после выполнения запроса

Синтаксис: GetDataTableAsync(string sqlQuertSelect)

Возвращаемое значение: таблица (DataTable)


Пример:

DataTable dataTable = null;

dataTable = await testerMySql.GetDataTableAsync("SELECT * FROM test_table");


foreach (DataRow row in dataTable.Rows)

                    foreach (DataColumn col in dataTable.Columns)

                            tester.ConsoleMsg(row[col].ToString());
