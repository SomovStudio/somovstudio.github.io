ConnectionOpenAsync

Описание: метод открывает соединение с базой данных MySql

Синтаксис: ConnectionOpenAsync(string connectionString)

Возвращаемое значение: отсутствует


Пример:

await testerMySql.ConnectionOpenAsync("server=127.0.0.1;uid=root;pwd=;database=test_db");
