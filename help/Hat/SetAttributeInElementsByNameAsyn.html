SetAttributeInElementsByNameAsync

Описание: метод вставить атрибут с указанным значением в множество элементов и в результате возвращает список

Синтаксис: SetAttributeInElementsByNameAsync(string name, string attribute, string value)

Возвращаемое значение: список (List)


Пример:

List<string> values = await tester.SetAttributeInElementsByNameAsync("link", "href", "www.test.ru");

foreach (string value in values)

{

       tester.ConsoleMsg(value);

}
