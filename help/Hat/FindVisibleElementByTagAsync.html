FindVisibleElementByTagAsync

Описание: метод выполняет поиск визуально отображаемого элемента с ожиданием в секундах и возвращает логический результат true или false

Синтаксис: FindVisibleElementByTagAsync(string tag, int index, int sec)


Пример:

bool result = await tester.FindVisibleElementByTagAsync("h1", 0, 5);
