GetCookiesAsync

Description: the method returns a list of cookies (document.cookie)..

Syntax: GetCookiesAsync()


Example:

List<string> cookies = await tester.GetCookiesAsync();

if (cookies.Count() > 0)

{

       foreach (string cookie in cookies)

       {

               tester.ConsoleMsg("COOKIE: " + cookie);

       }

}
