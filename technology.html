Технологии

В разработке программного обеспечения мы используем .NET Framework одну из самых передовых технологии от корпорации Microsoft.



Платформа Microsoft .NET Framework — это компонент Microsoft Windows, который пользователи могут получить с помощью различных каналов распространения, в число которых входят веб-узел Центра обновления Майкрософт и центр загрузки Microsoft.

.NET Framework — программная платформа, выпущенная компанией Microsoft в 2002 году. В настоящее время .NET Framework получает развитие в виде .NET Core, изначально предполагающей кроссплатформенную разработку и эксплуатацию.

ASP.NET — платформа разработки веб-приложений от корпорации Microsoft. ASP.NET входит в состав платформы .NET Framework.

Microsoft SQL Server — система управления реляционными базами данных, разработанная корпорацией Microsoft.
