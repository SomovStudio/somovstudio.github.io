Фреймворк: "AdminDatabase"

Данный фреймворк (платформа) позволяет создать администратор баз данных для Веб-приложения на основе Apache, MySQL с использованием языков программирования PHP и Javascript. Подобная "админка" часто используется для работы с данными в играх. Вы можете разработать свой администратор на основе данного фреймворка с последующим развертыванием на хостинге. Такая "админка" предоставит возможность работать с данными без использования phpmyadmin.

Системные требования:
Apache (version: 2.4.23 и выше.), MySQL (version: 5.7.14 и выше.), PHP (version: 5.6.25 и выше.) 		Архив приложения "AdminDatabase"
Руководство к приложениию "AdminDatabase" в формате PDF
Руководство к приложениию "AdminDatabase" в формате CHM

Программа: "FastADB"

Данная программа позволяет работать с Android Debug Bridge более удобно и быстрее. С помощью этой программы вы легко сможете снимать логи и скриншоты с устройства, устанавливать и удалять приложения, производить стресс-тестирвоание, и многое другое. Архив программы "FastADB"

Программа: "FastDatabase"

Программа для упрощенной работы с базами данных. С помощью этой программы вы можете создавать базы данных и таблицы в них, выбирать, добавлять, редактировать и удалять данные в таблицах. Программа работает с такими СУБД как: Microsoft SQL Server, Microsoft Access, MySQL. В дополнение присутствует удобное окно работы с XML. Архив программы "FastDatabase"

Программа: "FastNotepad"

Текстовый редактор представляет собой улучшенный блокнот с подсветкой кода и информационной базой данных. Архив программы "FastNotepad"
