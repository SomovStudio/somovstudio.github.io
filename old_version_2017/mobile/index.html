О компании

В нашем мире огромное количество информации, которую необходимо хранить и обрабатывать. Наша компания занимается разработкой:
Web-приложений, клиент-серверных приложения и проектированием баз данных.

Технологии

В данном разделе перечисленны основные технологие которые мы применяем в работе.

  • .NET Framework

    Программная платформа, выпущенная компанией Microsoft.

  • Yii2 Framework

    Объектно-ориентированный компонентный фреймворк, реализующий парадигму MVC

  • Bitrix Framework

    Система управления содержимым от российской компании 1С-Битрикс

  • Apache, MySQL, PHP

    Свободный веб-сервер, свободная реляционная система управления базами данных и скриптовый язык применяемый для разработки веб-приложений

  • Node.js

    Программная платформа, основанная на движке V8. Применяется преимущественно на сервере, выполняя роль веб-сервера.

  • jQuery

    Библиотека фокусирующаяся на взаимодействии JavaScript и HTML, помогает легко получать доступ к любому элементу DOM, предоставляет удобный API для работы с AJAX.

Услуги

Услуги в области разработки:
  • Web-приложений на основе Yii2, Bitrix.
  • Клиент-серверные приложений под Windows
  • Проектирование баз данных MySQL и Microsoft SQL Server
Внедрение и сопровождение:
  • Развертывание Web-приложения на платных хостингах с последующим сопровождением и продвижением в поисковых системах.
  • Полная поддержка разработанных Клиент-серверных приложений.
