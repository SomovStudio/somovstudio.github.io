Услуги

Мы предлагаем свои услуги в разработке под заказ программных средств любой сложности. В работе мы используем самые передовые .Net технологии разработанные корпорацией Microsoft.

Перечень услуг

Разработка программного обеспечения под заказ:

  • Клиент-серверные приложения под Windows
  • Клиент-серверные кроссплатформенные приложения
  • Web-приложения
  • Проектирование баз данных Microsoft SQL Server и MySQL

Внедрение и сопровождение:

  • Установка и настройка разработанного приложения
  • Развертывание разработанного Web приложения
  • Регулярная или частичная поддержка разработанного программного обеспечения
Расценки на услуги

Стоимость разработки:

  1. Приложение с локальной базой данных - 15000 р.
  2. Приложение с серверной базой данных - 25000 р.
  3. Кроссплатформенное приложение с серверной базой данных - 50000 р.
  4. ASP Net веб приложение с серверной базой данных - 35000 р.

Стоимость сопровождения:

  1. Регулярно - 2500 р. в месяц
  2. Почасовое сопровождение - 500 р. в час

* Цена может меняться в большую или меньшую сторону в зависимости от объема и сложности разрабатываемого проекта



Время на разработку:
В зависимости от сложности заказанного приложения, на его разработку может потребоваться от полу года до одного года.

Деятельность

Наша компания занимается проектированием баз данных и разработкой программных средств ориентированных на упрощенную работу с данными.
